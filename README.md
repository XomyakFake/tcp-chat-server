# TCP Multithreaded Chat Server

## Описание

Многопоточный TCP‑чат сервер, написанный на C++ с использованием POSIX sockets.  
Поддерживает одновременную работу нескольких клиентов, приватные сообщения и рассылку сообщений всем участникам.

Проект демонстрирует работу с сетевым программированием, многопоточностью и синхронизацией потоков.

---

## Функциональность

- Поддержка нескольких клиентов одновременно
- Рассылка сообщений всем пользователям (Broadcast)
- Приватные сообщения (@name)
- Потокобезопасная обработка клиентов (std::thread, std::mutex)
- Docker‑контейнеризация
- CI через GitHub Actions

---
